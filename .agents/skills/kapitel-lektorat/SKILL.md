# Kapitel-Lektorat
- **Stil**: Durchgehend freundliche "Du"-Form. Kurze, präzise Sätze ohne verschachtelten Nominalstil.
- **Didaktik**: Fachbegriffe erklären, Analogien nutzen. Klare Lernziele formulieren.
- **Code**: Keine fertigen Lösungen verraten. Nur unvollständige Code-Gerüste (`todo!()`) oder konzeptionelle Hinweise geben.
- **Format**: Rechtschreibung prüfen, Code-Elemente als inline-code formatieren.
