# Phase 7: Fearless Concurrency (Nebenläufigkeit)

In dieser Phase beschäftigen wir uns mit der parallelen Programmierung in Rust. Wir lernen, wie wir Code gleichzeitig ausführen können, ohne Datenrennen (Data Races) zu verursachen.

## Themen in dieser Phase
* **Threads & Shared State:** Wie man Threads startet und Daten sicher teilt.
* **Channels & Nachrichtenaustausch:** Kommunikation zwischen Threads mittels Message Passing.
* **Async/Await & Tokio-Grundlagen:** Asynchrone Programmierung für performante I/O-Operationen.
